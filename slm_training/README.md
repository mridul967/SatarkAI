# SatarkAI SLM Training Data & Fine-Tuning

This folder contains everything needed to fine-tune the local SLM.

## Files

| File | Purpose |
|------|---------|
| `training_data.jsonl` | Raw training pairs (gitignored — generated by the collector) |
| `validate_data.py` | Validates the JSONL before uploading to Colab |
| `SatarkAI_SLM_FineTune.ipynb` | Colab notebook (upload to Google Colab Pro) |

## Quick Start

```bash
# 1. Enable collection in .env
COLLECT_TRAINING_DATA=true

# 2. Run the app for 30-60 minutes to collect data
docker compose up

# 3. Validate
python slm_training/validate_data.py

# 4. Upload training_data.jsonl to Colab and run the notebook
```
